# UX Workspace

Use this structure for each project name (same folder name across sources):

- `Figma/<project>/pages/*.tsx`
- `Figma/<project>/mapping.json` (optional; page-to-slug map)
- `StitchDesktop/<project>/<slug>/code.html`
- `StitchMobile/<project>/<slug>/code.html`
- `Unified/<project>/` (generated)

Example:

- `Figma/rewardstracker/pages/Dashboard.tsx`
- `StitchDesktop/rewardstracker/dashboard/code.html`
- `StitchMobile/rewardstracker/dashboard/code.html`
